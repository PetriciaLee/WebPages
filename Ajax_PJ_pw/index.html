AJAX

AJAX nije programski jezik, već je kombinacija JavaScripta i HTML DOM-a. AJAX omogućuje asinkrono ažuriranje web stranice razmjenom podataka s web-poslužitelja iza scene. A to znači da je moguće ažurirati dijelove web stranice, bez ponovnog učitavanja cijele stranice. Ukratko, Ajax je JavaScript tehnologija koja klijentima štedi vrijeme.

Što možemo s AJAX-om?

  • Ažuriranje web stranice, bez ponovnog učitavanja stranice
  • Zatražiti podatke s poslužitelja - nakon što se stranica učita
  • Primanje podataka s poslužitelja - nakon što se stranica učita
  • Slanje podataka na poslužitelju - u pozadini

O AJAXU-u ukratko:

AJAX nije programski jezik, već je kombinacija JavaScripta i HTML DOM-a.

Ajax je JavaScript tehnologija koja klijentima štedi vrijeme.
